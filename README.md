# 🧠 How It Works

The game uses an AI-inspired elimination system:

1. User answers cricket-related questions
2. The engine filters IPL players dynamically
3. Confidence scores are updated after every answer
4. Candidate pool narrows progressively
5. Final prediction is generated intelligently
